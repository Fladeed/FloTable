# Changelog

All notable changes to FloTable with Views will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-20

### Added
- Initial release of FloTable with Views
- Core `TableWithViews` component with views system
- Generic `SimpleTable` component wrapper around ProTable
- `TableActions` component for custom toolbar actions
- TypeScript support with full type definitions
- Responsive design with mobile optimization
- Comprehensive documentation with Nextra
- Preview application with restaurant business example
- Support for custom views with filtering
- Built-in search functionality
- Pagination support
- Custom actions in toolbar
- Mobile-responsive design patterns

### Features
- **Views System**: Multiple predefined filtered views of data
- **Responsive Design**: Automatic mobile optimization
- **TypeScript**: Full type safety and IntelliSense support
- **Customizable Actions**: Add custom buttons to table toolbar
- **Search & Filter**: Built-in search with column filtering
- **Pagination**: Configurable pagination with size options
- **Internationalization**: i18n support through translation hooks
- **Ant Design Integration**: Built on Ant Design Pro Components

### Components
- `TableWithViews<T>` - Main table component with views
- `SimpleTable<T>` - Basic table wrapper
- `TableActions` - Toolbar actions component

### Hooks
- `useIsMobile` - Responsive design hook
- `useTranslation` - Internationalization hook

### Documentation
- Complete API reference
- Installation guide
- Quick start tutorial
- Customization examples
- Best practices guide
- Troubleshooting guide
- Mobile optimization guide

### Development
- Rollup build system
- TypeScript declaration files
- ESLint and Prettier configuration
- Comprehensive example application
- Git repository setup with documentation
